# harbourmaster
UI for Docker Engine API

Status: **pre-alpha**

# Development

```bash
$ pipenv shell
```

- Sample Docker URL via Flask app: ``http://localhost:9000/api/docker-engine/?url=/info``
- [API documentation](http://localhost:9000/api/doc/)

# Server

```bash
$ make run-server
```
